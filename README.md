# outersnarkness

Static landing page and interest form for the Outer Snarkness women's retreat,
a sister project of Ex-Mormon Snark. Single-file static HTML with embedded
brunch hero image, Jost typography, scroll-triggered card animations, and a
13-question retreat interest form.

Live: https://outersnarkness.com (Vercel)
Parent brand: https://exmormonsnark.com

## Stack

Plain static HTML. No build step. Vercel serves `index.html` directly.
